# ML From Scratch: A Personal Learning Journey

Welcome to my repository, where I document my journey through the core concepts of machine learning by implementing models and tools from scratch. This repository is both a personal reference and an educational resource for others interested in learning how machine learning works under the hood.

---

## Current Implementation

### Sequential Neural Network

- **Description:** A basic sequential neural network capable of multiclass classification.
- **Dataset:** Tested on the MNIST dataset using a 3 layer approach , with ReLu as the activation function for the hidden layer.
- **Features:** Implements forward propagation, backpropagation, and gradient descent from scratch using only Python and NumPy.
